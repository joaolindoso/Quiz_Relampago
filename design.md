# 🎨 Design System — Quiz Relâmpago

## 🎯 Objetivo Visual

Criar uma interface dinâmica, clara e moderna, com foco em velocidade e feedback imediato.

---

## 🎨 Paleta de Cores

| Função              | Cor       |
| ------------------- | --------- |
| Fundo               | `#f7f9ff` |
| Botão principal     | `#4a6cf7` |
| Hover do botão      | `#3d5be0` |
| Alternativa padrão  | `#ffffff` |
| Alternativa correta | `#b6f7c1` |
| Alternativa errada  | `#ffb3b3` |
| Barra de progresso  | `#4a6cf7` |

---

## 🖋 Tipografia

- Fonte: **Inter**, sans-serif
- Títulos: peso 600
- Alternativas: peso 500
- Texto geral: peso 400

---

## 📐 Espaçamentos

- Padding padrão: **16–24px**
- Espaço entre alternativas: **12px**
- Margem entre perguntas: **20px**

---

## 🧩 Componentes

### StartScreen

- Layout centralizado
- Botão com hover
- Marca discreta

### QuestionCard

- Caixa branca com sombra leve
- Alternativas com hover
- Estados:
  - normal
  - correta
  - errada

### ProgressBar

- Barra horizontal
- Animação suave

### EndScreen

- Pontuação grande
- Botão de reinício

---

## 🎬 Animações

- Fade da tela inicial: 0.5s
- Feedback de alternativa: 0.3s
- Barra de progresso: 0.4s

---

## 📱 Responsividade

- Pergunta centralizada
- Alternativas em coluna
- Botões grandes no mobile

---

## 🧭 Navegação

- Start → Game → End
- Reinício retorna ao Start

---

## 🧪 Acessibilidade

- Contraste adequado
- Botões com foco visível
- Texto claro e legível
